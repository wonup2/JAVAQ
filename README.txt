GitHub Pages Upload Instructions

Upload these files to one GitHub repository:

1. index.html
2. java-apcsa-assessment.html
3. java-code-tracing-20.html
4. .nojekyll

Then go to:
Settings > Pages > Build and deployment
Source: Deploy from a branch
Branch: main
Folder: / root
Save

Your site will usually appear at:
https://YOUR-GITHUB-ID.github.io/YOUR-REPOSITORY-NAME/

Important:
- Do not upload the answer key PDF to the public repository if students will take the quiz.
- The quizzes grade inside the browser only. Scores are not automatically saved.
