Java AP CS A Style Code Tracing Assessment

20 questions - loops, arrays, ArrayList, strings, and logic

Instructions: Choose the answer that shows exactly what is printed. Assume import java.util.*; and import java.util.Arrays; are available when needed. Assume each code segment is inside a valid main method unless another method is shown.

Suggested time: 40 to 50 minutes.

Question 1 - Loops

What is printed by the following code?

int sum = 0;
for (int i = 1; i <= 6; i++) {
    if (i % 2 == 0) {
        sum += i;
    } else {
        sum -= 1;
    }
}
System.out.println(sum);
